My article about the HTML <footer> element

The HTML <footer> element represents a small section on the very bottom of a webpage. Like the <header> which gives information to the user on the very beginning of a page, the footer's matter is for the visitors orientation. We mostly have a footer A <footer> element should contain information about its containing element.. You can have several <footer> elements in one document.

A HTML <footer> element typically contains:

  • authorship information
  • copyright information
  • contact information
  • sitemap
  • back to top links
  • related documents
